.. Installation

Installation
------------

PseudoNetCDF is easy to install using conda or pip.

conda
~~~~~

.. code-block:: bash

  conda install pseudonetcdf

pip
~~~

.. code-block:: bash

  pip install http://github.com/barronh/pseudonetcdf.git

Prerequisites
~~~~~~~~~~~~~

 * Required: numpy, netCDF4
 * Optional: matplotlib, pyproj, pandas
